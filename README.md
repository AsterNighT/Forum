## File Intro

### ./wwwroot
* /static  网站静态文件
  * /content  css文件
  * /images  网站的表情包，头图等图片文件 
  * /scripts 网站的所有脚本
  * /boardInfo.json 版面信息表
  * /98icon.ico 网站标识
  * /config.json 项目配置
  * /index.html 网站文件
  * /portrait.json 头饰配置
  
### ./Actions
react-redux的action

### ./AsyncActions
react-redux的异步action（用户中心）
